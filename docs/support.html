Pairing a LoRa radio

Innawoods works with any device flashed with the Meshtastic firmware (LilyGO T-Beam / T-Echo / T-Deck, Heltec, RAK Wireless). Flash with the official Meshtastic web flasher at flasher.meshtastic.org, set the channel + region in the official Meshtastic app, then in innawoods open Settings → LoRa radio → Set up step-by-step.

Going offline

To prepare for a trip without cell service:

  • Open Settings → Downloaded regions and download the region you'll be in.
  • Open the region's menu and tap "Download terrain (offline hillshade)" so the relief shading works without internet.
  • Pair your radio in Settings → LoRa radio.
  • Make sure every crew member is in the same party — share via the QR code on the party screen.
